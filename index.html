Tello drone swarm (3/4)

Opdrachtgever: Nova College Beverwijk
Tools: Python

Dit project heb ik tijdens mijn MBO opleiding in opdracht van mijn school uitgevoerd.
De opdracht was om voor een open dag 5 van de bovenstaande DJI Tello drones in formatie te laten vliegen met behulp van een router die voor de connectie heeft gezorgt tussen onze laptop en de drones.
De drones worden aangestuurd door een programma genaamd packetsender in combinatie met een python script die elke drone een specifieke opdracht kan laten uitvoeren.
